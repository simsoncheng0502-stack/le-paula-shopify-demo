# LE PAULA Shopify Demo

Public demo for a French-inspired luxury Shopify ecommerce concept.

Pages:

- `index.html` - homepage
- `blog.html` - journal / blog page
- `product.html` - product detail page
- `payment.html` - checkout demo page
- `thanks.html` - thank-you page

This is a static design prototype only. No real payment is processed.
